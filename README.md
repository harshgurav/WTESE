# BMI Calculator

A simple website that calculates BMI based on user input for weight and height.

## Files

- `index.html` - Main calculator page
- `style.css` - Black minimal styling
- `script.js` - BMI calculation logic
- `advice.html` - Tips/advice page based on BMI category

## How to Use

1. Open `index.html` in a web browser
2. Enter your weight in kg
3. Enter your height in cm
4. Click "Calculate" to see your BMI
5. Click "Get Tips / Advice" to view category-specific advice

## BMI Categories

- Underweight: BMI < 18.5
- Normal weight: BMI 18.5 - 24.9
- Overweight: BMI 25 - 29.9
- Obese: BMI ≥ 30
